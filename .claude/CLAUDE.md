# 开发团队角色协作系统

通过自定义命令实现高效的团队协作开发。

## 可用命令

| 命令 | 角色 | 用途 |
|------|------|------|
| `/项目经理` | 资深技术项目经理 | 需求分析、架构设计、任务拆解、技术选型 |
| `/程序员` | 高级全栈开发工程师 | 代码实现、开发计划管理 |
| `/测试员` | 高级 QA 工程师 | 代码审查、质量把控 |

## 快速开始

1. **新项目/新任务**: 输入 `/项目经理`，描述需求
2. **开始开发**: 项目经理完成设计后，输入 `/程序员`
3. **代码审查**: 程序员完成阶段后，输入 `/测试员`
4. **任务完成**: 全部审核通过后，输入 `/项目经理` 更新项目文档

## 目录结构

```
.claude/
├── CLAUDE.md              # 本文件
├── settings.local.json    # 本地权限配置
├── commands/              # 角色命令定义
│   ├── 项目经理.md
│   ├── 程序员.md
│   └── 测试员.md
├── docs/                  # 项目文档（由项目经理维护）
└── tasks/                 # 任务文档目录
```

## 重要规则

1. **沟通语言**: 所有角色与用户对话使用中文
2. **文件权限**: 各角色只能修改自己负责的文件
